# Docker Compose

## Containers

* mongodb, mongo-interface-express
* mysql, phpmyadmin
* nodejs, mongodb, mongo-interface-express
* postgreSQL, PgAdmin

***
## The start container
##### Para subir sem o modo daemon
`docker-compose up`\
##### Para subir em modo daemon
`docker-compose up -d`
##### Para subir junto com algum build
`docker-compose up -d --build`
